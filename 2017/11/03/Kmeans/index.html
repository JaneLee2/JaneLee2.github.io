Kmeans使用教程（Java版）

Kmeans是一种聚类算法，而聚类算法又隶属于非监督学习，即它可以将相似的对象归到同一个类中，即同类中的对象相似度很高，不同类中的对象相似度很低，有点像全自动分类技术。以下内容将主要介绍Kmeans算法的简单原理及库使用方法。

##一、Kmeans算法原理简介
Kmeans算法的名字（k、means）就已经总结了Kmeans算法的精髓思想，其一是它可以发现K个不同的簇群，其二是计算隶属该簇的所有对象均值作为该簇群的中心点。

1. 缺点：容易收敛到局部最小值、必须人为指定K值
2. 适用数据类型：数值型数据
3. 算法流程：

step1: 选取k值,可选距离指标。
step2: 随机确定K个初始点作为质心。
step3: 根据距离大小将数据集中的每个点分配到离其最近的簇中。
step4: 每个簇的中心更新为该簇所有点的平均值。
step5: 跳转至step3
step6: 判断：如果任意一个点的簇分配结果发生改变，跳转至step4，否则停止循环，结束聚类过程。

##二、Kmeans库使用方法
这一小节旨在通过一个小的demo例子来指导Kmeans算法的使用，主要涉及主要函数简介、demo、真实数据集验证。
1. 主要函数简介： 

1. kmeans.init(int k,double[][] items) 
   接收聚类个数K以及特征数据集，并随机初始化聚类中心以及聚类过程中用到的参数值。
2. kmeans.trainEclidAvgCenter()
   开始自动聚类过程：重复计算各个簇的中心更新中心点、划分各个对象所属簇类，直至停止。
3. kmeans.getClusterID_EculidDist(double[] item)  
   返回值：int，item的所属ClusterId
   即传入一个对象点，返回该对象所属的簇类号。
4. kmeans.getItemDist2ClusterId(double[] item)
   返回值：HashMap<Integer,Double>，key为item的所属ClusterId，value为item与所属ClusterId的距离值
   即传入一个对象点，返回该对象所属的簇类号以及与该簇类号的距离值。

2. demo：
这是撇开数据读入等复杂步骤，单看算法的一个demo，其中有诸多简化的地方。

1. import com.ruijie.rbds.machineLearningLibs.clustering.Kmeans;
2. //此处的numCluster、indexInfo权当满足要求的两个输入。
3. int numCluster = 3;

3. 真实小数据集验证效果：
接下来我们在一个小数据集Iris上Kmean算法的准确度，主要涉及数据集简介、数据准确率。
数据集简介：

Iris也称鸢尾花卉数据集，是一类多重变量分析的数据集。通过花萼长度，花萼宽度，花瓣长度，花瓣宽度4个属性预测鸢尾花卉属于（Setosa，Versicolour，Virginica）三个种类中的哪一类。

数据准确率： (50+47+36)/(50+5+50) = 85.8%
可以看出Setosa全对，而Versicolour将3个聚在了Virginica类中，而Virginica中将14个则聚在了Versicolour类中。
运行时间：4毫秒
综上，在实际大规模的数据上，兼准确率及运行时间综合考虑，是可以采用Kmeans算法的。
