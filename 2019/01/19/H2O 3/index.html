H2O

2019-01-19 • 分类: 机器学习平台

H2O.ai 是初创公司 Oxdata 在2014末推出的一个独立开源机器学习平台，主要服务于数据科学家和开发者，为其应用提供快速机器学习引擎。
其是一个分布式的、基于内存的、可扩展的机器学习和预测分析框架，适合在企业环境中构建大规模机器学习模型。它使用开发者熟悉的界面，可与 Hadoop 和 Spark 等大型数据技术无缝工作。它也提供许多流行算法的实现，例如 GBM、Random Forest、Deep Neural Networks、Word2Vec 等。

一、H2O发展史

1.2010年成立于加州山景城初创公司Oxdata开始H2O项目
2.2014年推出了H2O开源机器学习项目，获得890万美元的融资
3.2016年完成2000万美元的B轮融资
4.Gartner魔力象限对2016-2017年度数据分析平台的评测 第四象限
5.Gartner魔力象限对2018年度数据分析平台的评测 第一领导者象限

三、H2O Flow

在介绍H2O REST API之前先让我们认识一下H2O Flow，以下简称Flow，其是一个H2O web UI用来通过可视化调用restful API进行数据导入、训练模型、保存模型、用模型进行预测等操作，与此同时整个页面最小单元为Cell，可以对每个Cell进行注释，也可以如同工作流一样运行多个Cell，还可以注释与分享功能。

Flow支持REST API、R脚本和CoffeeScript，页面提供的图形化界面即使不会编程也能完成整个机器学习过程。

1.获取帮助

  1. 在Flow页面，单击 h 键可快速打开帮助快捷键

  2. Help>Assit Me 来获取帮助

  3. Flow 视频demo：view example Flows

  4. Flow 已安装的packs

  5. REST API文档

2.Flow Cell认识

Cell作为Flow的最小组成单元，数据导入、模型等也都是一个Cell。Cell有编辑和命令行模式。可以选择Cell的不同类型，编辑Cell的信息及其执行等。

命令行模式

  1. MD：Markdown
  2. CS:Code,新增一个Cell,默认就是CS cell
  3. RAW: Raw format 注释代码及一些评论
  4. H[1-6]: Heading level 标题等级

切换Cell类型

Cell类型用快捷键来切换，退出Cell的编辑模式，点击其左上角Cell类型的图标，输入快捷键。

选择Cell和运行Cell的菜单

  1. 菜单栏
  2. Cell 按钮

3.Flow CLIPS

如果把编辑好的单个Cell点击右方的回形针后，将会保存到CLIPS，方便后续重复利用此Cell。

  1. 保存：回形针
  2. CLIPS:我的Clips，系统自带，删除Clips，永久删除

4.Flow Outlines

Outlines可以看做当前Flow cells的一个内容大纲，可类比我们word中的目录导航功能

5.Flow 工作空间

整个Flow完成以后，可以将Flow保存下来，可以在右侧的FLOWS进行查看。
除保存操作外，Flow工作空间提供了复制、删除、导出Flow等功能，这些保存的Flow保存在硬盘上，启动的时候可以指定Flow路径读取。

6.数据源

从Flow页面来看，支持两种形式数据的导入：文件和jdbc方式，其中文件的方式是从服务器路径上读取，能支持的格式如下：

AUTO

XLS (BIFF 8 only)
XLSX (BIFF 8 only)

一般流程为：import files读取本地文件 -> import 成为.hex文件 -> setup parse（解析）数据 -> parse数据 -> getFrameSummary信息（获取统计信息）

大体流程如下：构建预测（选择训练好的模型、需要预测的数据）-> 开始预测（预测程序运行,输出结果）

4.导入数据和解析数据API

GET /3/ImportFiles
Import raw data files into a single-column H2O Frame.

POST /3/ParseSetup
Guess the parameters for parsing raw byte-oriented data into an H2O Frame.

POST /3/Parse
Parse a raw byte-oriented Frame into a useful columnar data Frame.

5.Frames API

GET /3/Frames
Return all Frames in the H2O distributed K/V store.

GET /3/Frames/(?.*)
Return the specified Frame.

GET /3/Frames/(?.*)/summary
Return a Frame, including the histograms, after forcing computation of rollups.

GET /3/Frames/(?.*)/columns/(?.*)/summary
Return the summary metrics for a column, e.g. mins, maxes, mean, sigma, percentiles, etc.

DELETE /3/Frames/(?.*)
Delete the specified Frame from the H2O distributed K/V store.

DELETE /3/Frames
Delete all Frames from the H2O distributed K/V store.

五、H2O-3 Python

H2O 有for FLOW UI ，for R，for Python 版本，可以将H2O看作一个自我封装封闭的机器学习环境，虽然其内部是Java所写，但非常友好的提供了Python的API，可以在Python中无缝使用H2O的方法。

与scikit-learn联合使用

在H2O官方2018年发布的第五版《Machine Learning with Python and H2O》中提及H2O与scikit-learn算法一起使用的两个例子：Pipelines和Randomized Grid Search，比如下：

from sklearn.grid_search import RandomizedSearchCV   
from h2o.cross_validation import H2OKFold   
from h2o.model.regression import h2o_r2_score   
from sklearn.metrics.scorer import make_scorer    
params = {"standardize__center": [True, False],     
          "standardize__scale": [True, False],   
          "pca__k": [2,3],   
          "gbm__ntrees": [10,20],   
          "gbm__max_depth": [1,2,3],   
          "gbm__learn_rate": [0.1,0.2]}   
custom_cv = H2OKFold(iris_df, n_folds=5, seed=42)   
pipeline = Pipeline([("standardize", H2OScaler()),    
                     ("pca", H2OPrincipalComponentAnalysisEstimator(k=2)),   
                     ("gbm", H2OGradientBoostingEstimator(distribution="gaussian"))])   
random_search = RandomizedSearchCV(pipeline, params,   
                                   n_iter=5,   
                                   scoring=make_scorer(h2o_r2_score),    
                                   cv=custom_cv,    
                                   random_state=42,    
                                   n_jobs=1)   
random_search.fit(iris_df[1:], iris_df[0])
