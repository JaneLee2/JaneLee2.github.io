Linux Python3.6 安装

2019-01-19 • 分类: Python 安装

Python3

Centos7默认自带Python是2.7.5，用到的fbprophet算法需求的Python3（gcc版本要求高）。因此在Centos7 上安装两个Python：Python2和Python3

物理机安装

  1. 采用rpm包安装Python3 

    yum install -y https://centos7.iuscommunity.org/ius-release.rpm   
yum makecache   
yum install -y python36u python36u-libs python36u-devel python36u-pip   
python3.6 -V

  2. 安装更新gcc（4.8.5）

    yum install gcc gcc-c++

  3. 安装其他所需包 

    pip3.6 install pandas   
pip3.6 install fbprophet
pip3.6 install pymongo
pip3.6 install scikit-learn
