#windows安装fbprophet
在windows上安装fbprophet最开始先尝试python或winpython，再使用pip安装fbprophet，遇到诸多问题，后边看到作者建议使用anaconda作为Python解释来安装，下面将大致步骤记录如下：

系统

windows7 64位
python3.5 visual studio 2015
python3.6 visual studio 2017

第一步 C++

安装C++编译器Microsoft Visual C++ Build Tools

第二步 Anacoda

  1. 安装Anaconda（注意安装路径不要有空格）
  2. 添加国内清华镜像
    1） 从开始菜单打开anaconda的客户端

    2）运行 conda config –add channels https://mirrors.tuna.tsinghua.edu.cn/anaconda/pkgs/free/
    3) 运行 conda config –add channels https://mirrors.tuna.tsinghua.edu.cn/anaconda/cloud/conda-forge/
    4）运行 conda config –add channels https://mirrors.tuna.tsinghua.edu.cn/anaconda/cloud/msys2/
    5）conda config –set show_channel_urls yes
    6）使用everything找到.condarc文件目录（默认在C:\Users\用户名XXX.condarc）将channels的defaults行删除，保存。
    7）运行conda config –show查看配置

    第三步 Gcc

    安装gcc编译器

第四步 fbprophet

conda install -c conda-forge fbprophet

第五步 pycharm中引入anaconda的Python解释器
