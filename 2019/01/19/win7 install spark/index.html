win7 spark安装

2019-01-19 • 分类: Spark

最近需要在windows上测试spark的MLlib和其Dataframe，为了在pycharm中调试方便，便想在windows上安装spark，下面将大致步骤记录如下：

系统

windows7 64位
Java jdk1.8
scala 2.11.8
spark 2.x 因为1.x版本不支持Dataframe

安装 java

安装略
cmd输入 java -version查看版本和配置是否正确

安装 scala

安装略
cmd输入 scala -version查看版本和配置是否正确

安装 spark

1.下载http://spark.apache.org/downloads.html
2.解压即安装（可重命名）
3.配置spark的环境变量 
  path=D:\spark\bin

安装 hadoop

1.下载https://archive.apache.org/dist/hadoop/common/(这里要和spark中选择的hadoop保持一致)
2.解压即安装（可重命名）
3.配置hadoop的环境变量
  D:\hadoop\bin

安装 winutil

winutil软件的作用是为了hadoop可以兼容在windows下运行，在github上可以找到很多版本。
1.下载https://github.com/steveloughran/winutils(这里要和hadoop的版本保持一致)
2.将winutils.exe拷贝到hadoop的/bin目录下
3.执行winutils.exe chmod -R 777 D:/tmp/hive

测试spark是否安装成功

1.进入cmd,运行spark-shell
2.等待出现scala的spark-shell即成功
